## Решение задачи y' = A*y(x) явным и неявным методами , где A - матрица m x m, y(x) - вектор размерности m.
### Схема Бутчера для явного метода:
     0|  0   0
   2/3| 2/3  0
      |---------
      | 1/4 3/4
### Схема Бутчера для неявного метода:
   1/4| 1/4  0
   3/4| 1/2 1/4
      |---------
      | 1/2 3/2
